🚀 Collections

Document => Mongo stores data in form of documents (BSON docs)

Collection => MongoDB stores documents in collections


🚀 Collections (Anime Style)

Document 👉
MongoDB data ko documents ke form me store karta hai (BSON format me).
Document matlab ek character ka complete data.

🧠 Example (Anime):
Socho Naruto ek anime hai.
Naruto ka data ek document hoga, jaise:

Name: Naruto
Village: Leaf
Power: Rasengan
Rank: Hokage

➡️ Ye sab milkar ek single document ban gaya (Naruto ka record).


Collection 👉
MongoDB bahut saare documents ko milakar collection banata hai.
Collection matlab sab characters ka group.

🧠 Example (Anime):
Naruto ka data = 1 Document
Sasuke ka data = 1 Document
Sakura ka data = 1 Document

➡️ In sab characters ke documents milkar jo banta hai = Collection

👉 Short me:

Document = ek anime character ka full data

Collection = sab anime characters ka group

🎯 Easy line:

Naruto = Document

Naruto, Sasuke, Sakura = Collection
